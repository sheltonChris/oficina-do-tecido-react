# Oficina do Tecido React

## Descrição
Projeto em React que simula uma loja de tecidos com funcionalidades de visualização de produtos, interação básica e navegação.

Este projeto demonstra minhas habilidades com React, componentes, estado e organização de código.

## Funcionalidades
- Listagem de produtos
- Navegação entre telas
- Uso de componentes funcionais e props

## Tecnologias
- React
- JavaScript
- CSS
